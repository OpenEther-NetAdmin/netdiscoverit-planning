## About

NetDiscoverIT solves the problem of stale network documentation by automatically discovering devices, collecting configurations, and using AI to generate living documentation and recommendations.

**Key Features:**
- Privacy-first (local agent, data stays on-prem)
- AI-powered documentation
- ML-based device classification
- Proactive recommendations
- Human-in-the-loop changes
